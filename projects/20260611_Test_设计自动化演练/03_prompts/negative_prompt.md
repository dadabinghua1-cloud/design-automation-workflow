# 负面提示词模板

用途：统一记录所有底图延展时需要避免的内容。当前阶段只作为人工检查和提示词参考，不接入任何生成工具。

## 1. 统一负面提示词

不要乱码文字，不要不可读文字，不要模拟文字，不要二维码，不要箭头，不要地址，不要多余 Logo，不要重复 Logo，不要卡通，不要二次元，不要促销风，不要电商大促风，不要复杂粒子爆炸，不要过度炫光，不要杂乱，不要低清晰度，不要模糊，不要噪点，不要压缩痕迹，不要主体变形，不要比例错误，不要破坏主 KV 风格。

## 3. 底图专用限制

- 底图中只保留背景、氛围、装饰和必要的主视觉元素。
- 正式文案、二维码、箭头、地址、Logo 默认后期添加。
- 不要把预留区填满。
- 不要让背景元素影响后期排版。
- 不要生成难以修改的复杂叠加内容。
