# 主风格提示词模板

用途：统一所有延展底图的视觉方向。当前阶段只作为提示词模板和人工说明，不接入任何生成工具。

## 1. 项目信息

- 项目名称：
- 主 KV 路径：`00_input/kv/`
- 参考图路径：`00_input/references/`
- 输出物料清单：`02_material_list/material_list.csv`
- 是否只做底图：默认是

## 2. 主风格原则

- 基于主 KV 进行视觉延展，保持品牌识别、主色调、视觉气质和空间氛围一致。
- 整体风格应高级、简约、商业化、干净、有留白。
- 默认只制作底图，不直接生成最终文字排版。
- 文字、二维码、箭头、地址、Logo 默认后期添加。
- 画面需要保留清晰的可编辑空间，方便后续在 PS / Figma 中修改。
- 背景层次要清晰，不要让装饰元素抢占核心内容区。
- 横版、竖版、不同尺寸之间应保持同一套视觉语言。

## 3. 可复用主提示词

请基于主 KV 进行设计延展，生成适合多尺寸商业活动物料使用的高级简约底图。画面需要延续主 KV 的品牌色、视觉氛围、材质质感和构图逻辑，整体干净、克制、商业化，有清晰留白和可放置文字的区域。

默认只生成底图，不要直接生成正式文字、二维码、箭头、地址或多余 Logo。画面应方便后续在 PS / Figma 中添加文字信息、Logo、二维码和导视元素。构图需要适配指定尺寸，主体不应被裁切，背景不应杂乱，整体风格应统一、清晰、高级。

```text
需要预留的位置：
输出说明：只做底图，文字 / Logo / 二维码 / 箭头 / 地址后期添加。
```
